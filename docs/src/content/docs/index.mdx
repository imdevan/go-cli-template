## Quick Start

```bash
# Build and run
just build-run

# View configuration
go-cli-template config

# Generate shell completion
go-cli-template completion bash > /etc/bash_completion.d/go-cli-template
```

## Architecture

The project follows a clean architecture pattern:

- **cmd/** - CLI entrypoint and commands
- **internal/config** - Configuration management
- **internal/domain** - Domain models
- **internal/ui** - Bubble Tea UI components
- **internal/utils** - Utility functions
- **internal/adapters** - External service adapters
